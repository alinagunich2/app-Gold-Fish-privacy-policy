Политика конфиденциальности

Настоящая политика описывает, как мобильное приложение FishGold обрабатывает данные при его использовании. Приложение предназначено для планирования рыбалки: пользователь сохраняет места на карте, по координатам этих мест приложение запрашивает погодные данные и показывает расчётный прогноз клёва по видам рыб; интервал и шаг показа (в том числе почасово) зависят от версии приложения и выбранного тарифа.

1. Общие положения

Политика применяется к мобильному приложению FishGold. Приложение помогает пользователю сохранять точки рыбалки, просматривать карту, получать прогноз погоды по выбранной точке и видеть расчётный прогноз клёва по отдельным видам рыб.

Доступ к части функций приложения может зависеть от выбранного тарифа.

2. Какие данные обрабатываются

Состав данных зависит от того, какие функции приложения вы используете и какие разрешения предоставляете.

  1. Сохранённые точки на карте. Координаты выбранных пользователем мест и связанные с ними данные, например название точки.
  2. Геолокация устройства. При предоставлении разрешения приложение может использовать текущее местоположение пользователя для работы карты и выбора мест рыбалки. Доступ к геолокации можно отключить в настройках устройства.
  3. Погодные данные. По координатам сохранённой точки приложение запрашивает погодную информацию у внешнего поставщика погодных данных.
  4. Данные о подписке. При оформлении подписки могут обрабатываться сведения о статусе доступа, сроке действия подписки и технические идентификаторы покупки.
  5. Технические данные. При работе приложения могут обрабатываться сведения, необходимые для загрузки карты, выполнения сетевых запросов, обеспечения стабильности и безопасности работы.
  6. Локально сохранённые данные. Часть информации хранится локально во внутреннем хранилище приложения на устройстве пользователя, включая координаты, параметры точек, загруженные погодные данные и данные, необходимые для отображения прогноза клёва.

3. Внешние сервисы

Для работы отдельных функций приложение может использовать сторонние сервисы и платформы.

  • Картографический сервис — для отображения карты, маркеров и взаимодействия с географическими объектами.
  • Поставщик погодных данных — для получения погодной информации по координатам выбранной точки.

Указанные сторонние сервисы могут обрабатывать данные в соответствии со своими собственными политиками конфиденциальности.

4. Передача данных

Мы не продаём персональные данные пользователей. Передача данных третьим лицам возможна только в случаях, когда это необходимо для работы функций приложения или требуется по закону.

  • Координаты точки могут передаваться сервису погоды для получения прогноза.
  • Данные, связанные с отображением карты, могут обрабатываться картографическим сервисом.
  • Данные, связанные с подпиской, могут обрабатываться магазинами приложений (оформление, продление и учёт покупок).
  • Для проверки и синхронизации статуса подписки технические данные покупки и доступа могут передаваться и обрабатываться сторонним сервисом управления подписками, через который приложение сверяет статус доступа с магазинами приложений.

5. Хранение данных

Значительная часть данных хранится локально на устройстве пользователя во внутреннем хранилище приложения. Это относится, в частности, к сохранённым координатам, параметрам точек, загруженным погодным данным и данным, используемым для отображения прогноза клёва.

Данные хранятся столько, сколько это необходимо для работы приложения, если более длительный срок не требуется законодательством или правилами сторонних платформ.

При удалении приложения локально сохранённые данные обычно удаляются вместе с приложением, однако история покупок и подписок может храниться у Apple, Google или иных сервисов по их собственным правилам.

6. Права пользователя

В зависимости от применимого законодательства пользователь может иметь право запросить информацию об обработке данных, их исправление, удаление, ограничение обработки, а также отозвать ранее предоставленные разрешения, например доступ к геолокации.

По вопросам, связанным с обработкой данных, вы можете написать на адрес supportFishGold@gmail.com.

7. Несовершеннолетние

Приложение не содержит контента для взрослых и может использоваться широкой аудиторией. Однако оформление подписок, платежей и иные действия, для которых по закону требуется согласие родителей или законных представителей, должны осуществляться с соблюдением применимого законодательства и правил платформы.

8. Изменение политики

Мы можем обновлять настоящую политику при изменении функций приложения, состава используемых сервисов или требований законодательства. Актуальная версия политики публикуется по тому же адресу, где размещён этот документ.
